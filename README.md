# 番茄旅行AI

智能旅游助手，让旅行更简单 🍅

## 功能特点

- 🤖 **AI智能匹配** - 输入需求自动匹配最合适的供应商
- 🏢 **供应商管理** - 浏览、搜索、筛选供应商
- 💬 **实时沟通** - 与供应商在线沟通
- 📋 **行程规划** - 自动生成行程方案

## 技术栈

- 纯前端实现 (HTML + CSS + JavaScript)
- localStorage 本地存储
- 响应式设计
